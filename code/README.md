# ContextBrief Frontend

Фронтенд проекта **ContextBrief** на Vue 3.

## Что внутри

- стартовая страница с поиском темы сразу после открытия;
- красивый блок результата с резюме, тезисами, рекомендациями и источниками;
- переключение темы оформления;

## Запуск

```bash
npm install
npm run serve
```

По умолчанию приложение открывается на `http://localhost:8081`.

## Переменные окружения

- `VUE_APP_ML_API_BASE_URL` — прямой адрес Python-сервиса, если не используется proxy.
- `VUE_APP_ML_PROXY_TARGET` — адрес Python-сервиса для dev proxy, по умолчанию `http://localhost:5000`.
